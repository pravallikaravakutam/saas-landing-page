# CloudSync SaaS Landing Page

## Project Overview

CloudSync is a modern SaaS Landing Page developed using HTML5, CSS3, Bootstrap 5, and CSS Grid. The project demonstrates responsive web design principles and professional UI layout creation.

## Objectives

* Design a modern SaaS landing page
* Create responsive layouts
* Implement Bootstrap 5 components
* Practice CSS Grid and UI alignment
* Build reusable UI sections

## Technologies Used

* HTML5
* CSS3
* Bootstrap 5
* CSS Grid
* Google Fonts (Poppins)

## Features Implemented

### Navigation Menu

Responsive Bootstrap navigation bar with mobile menu support.

### Hero Banner

Professional hero section with:

* Gradient background
* Call-to-action buttons
* Responsive image

### Features Section

* CSS Grid layout
* Hover effect cards
* Responsive design

### Pricing Cards

* Three pricing plans
* Featured plan highlight

### Testimonials

* Customer feedback cards
* Responsive card layout

### FAQ Section

* Bootstrap Accordion component

### Contact Section

* Responsive contact form
* User-friendly design

### Footer

* Company information
* Copyright section
